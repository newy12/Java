# final_19oclock

### 팀명/팀원 소개
``` 
> 팀명 | 디벨로퍼되고시퍼
      좋은 개발자가 되고 싶은 마음을 담았습니다.
> 팀원 | 김영재, 이다솜, 이도현, 임소현, 최문정
      김영재 : 프리랜서 소개페이지, 마이페이지, 서비스 등록
      이다솜 : 서비스 검색/조회, 서비스 상세페이지, 공지사항
      이도현 : 헤더/푸터, 회원가입, 로그인, 메인페이지, 문의사항
      임소현 : 채팅, 관리자-이용자관리, 서비스관리
      최문정 : 이용자 마이페이지, 결제내역 관리, 장바구니, 의뢰게시판
```

### 프로젝트 소개
``` 
> 진행 기간 | 2020년 12월 24일 ~ 2021년 01월 25일

> 프로젝트 이름 | 19시:일구하고시퍼
      프리랜서가 개인의 노동력을 등록하면, 필요한 이용자가 등록된 서비스를 조회하고 쉽게 이용할 수 있게 도와주는 프리랜서 마켓을 제작
      
> 기획 의도  
      최근 코로나19로 인한 일자리 감소과 언택트 근무 형태의 증가되어 프리랜서 마켓이 확산되고있다고 판단하여 기존 사이트의 단점을 보완하고 기능을 확장한 사이트를 제작
      
> 개발 목표 및 주안점
      확장성 : 사소한 서비스도 등록 가능하게 하여 사용자의 범위를 확장하고, 접근성을 높
      신뢰 : 서비스를 등록할 떄 관리자의 승인이 필요하도록 하여, 선별된 글을 확인할 수 있음
      비교/검색 : 이용자가 원하는 서비스를 한눈에 확인할 수 있는 카테고리 제공, 정확한 키워드로 검색하지 않아도 서비스 검색이 가능

> 개발 환경
      협업도구 : Github, Sourcetree
      개발도구 : Spring, Brackets
      DB : Orale, MyBatis
      개발언어 : Java, JavaScript, jQuery, HTML5, CSS3
      서버 : Apache Tomcat 8.5
      
```

### 기능 소개
``` 
> 서비스 이용자(의뢰인)
    1. 서비스 조회
     - 카테고리를 통한 서비스 조회
     - 검색 키워드를 통한 서비스 검색
     - 서비스 세부 페이지 확인
    2. 찜 하기
     - 마음에 들거나 관심있는 서비스를 저장하는 기능
    3. 의뢰글 작성
     - 원하는 서비스가 없으면 직접 의뢰글을 작성
    4. 결제
    - 결정된 서비스를 결제
    5. 채팅
     - 프리랜서에게 문의할 내용 또는 작업 진행 내용을 전달할 때 채팅 이용
    6. 이용후기/평점
     - 거래가 끝나면 해당 프리랜서 게시글에 이용후기/평점을 등록

> 서비스 판매자(프리랜서)
    1. 서비스 글 작성
      - 제목,카테고리, 지역,가격, 작업기간 등 내용을 기입
      - 등록을 하면 관리자에게 업로드 요청
    2. 알림 기능
      - 사용자에게 채택된 경우 알림
      - 관리자의 서비스 승인 알림 메시지
      - 의뢰인의 게시글에 후기가 등록된 경우
    3. 채팅 기능
      - 의뢰인과 정보를 소통할 수 있도록 채팅 기능
    4. 마이페이지
      - 프리랜서 자기소개 관리
      - 작성한 서비스 글 관리

> 사이트 관리자
    1. 이용자 관리(의뢰인 관리)
      - 프리랜서들의 평가를 기준으로 좋지 않은 이용자를 관리
      - 회원에게 메시지 
    2. 제공자 관리(프리랜서 관리)
      - 프리랜서가 서비스 등록 하면 관리자가 확인하고 승인
    3. 게시판 관리
      - 공지사항 게시글 작성/수정/삭제
      - 고객문의 문의글 답변 작성/수정/삭제 기능
      
```
